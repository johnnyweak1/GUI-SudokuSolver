
# Sudoku Solver

Sudoku Solver desktop app based on backtracking algorithm.



## Run Locally

Clone the project

```bash
  git clone https://github.com/johnnyweak1/GUI-SudokuSolver
```

Go to the project directory

```bash
  cd GUI-SudokuSolver
```
Start script

```bash
  python3 main.py
```
